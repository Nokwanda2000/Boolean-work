<h1>Boolean-work</h1> 

## Run Locally
Clone the project
```bash
  git clone https://github.com/Nokwanda2000/Boolean-work.git
```
Go to the project directory
```bash
  cd desktop
```
Install dependencies
```bash
  npm install
```
Start the react server
```bash
  npm run dev
```
## Tech Stack
**Client:** React JS, HTML5, JavaScript, CSS3
